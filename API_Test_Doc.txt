----------------------------------------------------------------------
Test Scenarios
----------------------------------------------------------------------
1. General Tests (Applicable to ALL APIs)

Verify API returns 200 OK for a valid request.
Verify API returns 401 Unauthorized when API key is missing.
Verify API returns 401 Unauthorized when API key is invalid.
Verify API returns 404 Not Found for invalid endpoint.
Verify API returns 400 Bad Request for invalid query parameters.
Validate response JSON structure matches the expected TMDB schema.
Validate required fields exist in every result (e.g., id, title, overview, genre_ids).

3. Pagination Tests

Verify page=1 returns results.
Verify page=n returns correct page number.
Verify page 0 returns error (boundaries).
Verify invalid page values (page=-1, page=10000).
Validate total_pages and total_results exist.
Verify navigating through pages returns different content.

4. API Key Tests (-ve)

Verify response when API key is empty.
Verify response when API key has special characters.

5. Sorting Tests (For discover APIs)

discover_newest: Validate results are sorted by release_date descending.
discover_top_rated: Validate results are sorted by vote_average descending.
discover_trend: Validate results are sorted by vote_count descending.
discover_popular: Validate results are sorted by popularity descending.

6. Filter Tests (for discover APIs)
Year Range Filters
    Validate filtering works for release_date.gte lower bound.
    Validate filtering works for release_date.lte upper bound.
    Validate invalid date format returns error.
Rating Filters
    Validate vote_average.gte applied correctly.
    Validate vote_average.lte applied correctly.
    Validate invalid values (e.g., rating = 100).
Genre Filters
    Validate with_genres=28 only returns movies with genre 28 (Action).
    Validate invalid genre returns empty list.

7. Specific API Functional Tests
A. MOVIE APIs
popular_movie
    Validate movies are sorted by popularity (descending).
    Validate results contain popular movies as per popularity.

trending_movie
    Validate trending movies appear in the list.
    Validate results has trending movie/week.

newest_movie
    Validate newly released movies appear.
    Validate recent release dates.

top_rated_movie
    Validate high rated movies appear.
    Validate vote_average sorted descending.

search_movie
    Validate movies matching for valid query are present.
    Validate search returns empty list for random/invalid query.
    Validate error with special character queries (e.g., "!@#&").

B. TV APIs

trending_tv

newest_tv

top_rated_tv

search_tv
